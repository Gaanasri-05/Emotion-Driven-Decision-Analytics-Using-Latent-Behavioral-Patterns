# Emotion-Driven Decision Analytics Using Latent Behavioral Patterns

An AI/ML-based project focused on analyzing emotional and behavioral patterns to study decision-making trends using machine learning and data analytics techniques.

## Features
- Data preprocessing and cleaning
- Text and behavioral pattern analysis
- Feature extraction using TF-IDF
- Machine learning model training
- Prediction and evaluation
- Emotion-driven decision analysis

## Technologies Used
- Python
- Machine Learning
- Scikit-learn
- Pandas
- NumPy
- TF-IDF Vectorization
- Jupyter Notebook

## Project Structure
- `data/` → Dataset files
- `notebooks/` → Experiment notebooks
- `src/` → Source code modules
- `results/` → Trained models and vectorizers

## Modules
- Preprocessing
- Feature Extraction
- Model Training
- Evaluation
- Prediction

## Machine Learning Workflow
1. Data Collection
